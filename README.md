# DGSW_Seminar_community
대구소프트웨어고등학교 '학생 전문가 특강' 을 위한 커뮤니티

> 선후배간의 지식 전달과 교내에 자율적인 학습을 위하여 개발되었으며,     
> 강의 목록 페이지에 자신이 가르치고 싶은 분야를 올리고 강의 요청 페이지에
> 듣고싶거나 바라는 강의를 적어주시면 됩니다 !


# 기술 스택
 * ReactJS
 * Django
 * JWT
 * Bootstrap
 * GraphQL(Graphene)
